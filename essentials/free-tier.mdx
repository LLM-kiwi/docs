## Ideal Use Cases

The free tier is designed for:

- **Prototyping** new AI-powered features
- **Learning** the API and exploring capabilities  
- **Small projects** with modest usage requirements
- **Testing** before committing to a paid plan

## Daily Reset

All usage limits reset automatically at **Midnight UTC** (00:00 UTC) each day. There's no rollover of unused requests.
