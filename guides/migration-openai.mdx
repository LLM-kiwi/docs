llm.kiwi is designed as a drop-in replacement for the OpenAI API. Migrating requires minimal code changes‚Äîoften just updating the base URL.

## Quick Migration

### Python

```python Before (OpenAI)
from openai import OpenAI

client = OpenAI(
    api_key="sk-..."
)
```

```python After (llm.kiwi)
from openai import OpenAI

client = OpenAI(
    base_url="https://api.llm.kiwi/v1",  # Add this line
    api_key="your-llm-kiwi-key"           # Use llm.kiwi key
)
```

### Node.js

```javascript Before (OpenAI)
import OpenAI from 'openai';

const client = new OpenAI({
  apiKey: "sk-..."
});
```

```javascript After (llm.kiwi)
import OpenAI from 'openai';

const client = new OpenAI({
  baseURL: 'https://api.llm.kiwi/v1',  // Add this line
  apiKey: process.env.LLM_KIWI_API_KEY  // Use llm.kiwi key
});
```

## Model Mapping

Map OpenAI models to llm.kiwi equivalents:

| OpenAI Model | llm.kiwi Equivalent | Notes |
| :--- | :--- | :--- |
| `gpt-4` | `gpt-oss-20b` | High capability open variant |
| `gpt-4-turbo` | `gpt-oss-20b` | Balanced performance |
| `gpt-3.5-turbo` | `default` | Fast routing |
| `gpt-4o-mini` | `default` | Model-lite routing |

<Tip>
  Use llm.kiwi's `default` model to let our router automatically select the best model for each request.
</Tip>

## Environment Variables

Update your environment configuration:

```bash Before
OPENAI_API_KEY=sk-...
OPENAI_BASE_URL=https://api.openai.com/v1
```

```bash After
LLM_KIWI_API_KEY=your-key-here
LLM_KIWI_BASE_URL=https://api.llm.kiwi/v1
```

## Framework-Specific Guides

### LangChain

```python Python
from langchain_openai import ChatOpenAI

llm = ChatOpenAI(
    base_url="https://api.llm.kiwi/v1",
    api_key="your-llm-kiwi-key",
    model="gpt-oss-20b"
)
```

### LlamaIndex

```python Python
from llama_index.llms.openai import OpenAI

llm = OpenAI(
    api_base="https://api.llm.kiwi/v1",
    api_key="your-llm-kiwi-key",
    model="gpt-oss-20b"
)
```

```typescript TypeScript

const llmkiwi = createOpenAI({
```

## Testing Your Migration

Verify your migration works correctly:

```python Python
# Test basic functionality
response = client.chat.completions.create(
    model="default",
    messages=[{"role": "user", "content": "Hello, is this working?"}]
)
print("‚úÖ Basic request:", response.choices[0].message.content[:50])

# Test streaming
stream = client.chat.completions.create(
    model="default",
    messages=[{"role": "user", "content": "Count to 3"}],
    stream=True
)
print("‚úÖ Streaming:", end=" ")
for chunk in stream:
    if chunk.choices[0].delta.content:
        print(chunk.choices[0].delta.content, end="")
print()

print("üéâ Migration successful!")
```

## Common Issues

OpenAI model names don't work directly. Use llm.kiwi models:
- `gpt-4` ‚Üí `gpt-oss-20b` (or other Pro models)
- `gpt-3.5-turbo` ‚Üí `default`
